# VibePlay

Site simples de entretenimento feito com HTML, CSS e JavaScript.

Funciona direto no GitHub Pages porque não usa PHP, banco de dados ou framework.

## Como publicar no GitHub Pages

1. Deixe os arquivos `index.html`, `style.css` e `script.js` na raiz do repositório.
2. Faça commit e push.
3. No GitHub, vá em Settings > Pages.
4. Escolha Deploy from a branch, branch `main`, pasta `/root`.

O ranking e o mural usam `localStorage`, então ficam salvos no navegador de cada visitante.
