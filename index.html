Como funciona

Sem login, sem instalação, só diversão rápida.

Tudo funciona localmente no navegador com JavaScript e Local Storage. Ranking, tema e recados ficam no próprio dispositivo.

  1. 1

    Escolha uma experiência

    Entre pelo jogo, pelo quiz ou pelo mural.

  2. 2

    Jogue, responda ou escreva

    Use poucos minutos para participar do seu jeito.

  3. 3

    Veja o resultado salvo no navegador

    Os dados ficam guardados localmente, sem envio para servidor.
